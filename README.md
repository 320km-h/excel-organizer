# 陈列照片整理工具

> Windows 小工具：把门店陈列照片（嵌在 Excel 单元格里）自动按 **货架 / 冰箱 / 地堆** 分类、按拜访档期归位，填进模板生成成品 Excel。

## 这是什么

快消 / 零售业务里，业务代表常把每家门店的陈列照片拍下来塞进一个 `.xlsx`。本工具读取这些嵌入图片，用 AI 视觉模型自动判类（货架 / 冰箱 / 地堆）、去重、按"上半档 / 下半档"归位，最后填进标准模板，生成可直接交付的整理结果。

整个过程一键完成，不需要懂编程。

## 适用环境

- **Windows 10 / 11（64 位）**
- 无需安装 Python —— 发布包已自带运行环境
- 需要一个阿里云通义千问 **DashScope API Key**（自费；分类调用付费视觉模型，用量很小，约千张图几元人民币）

## 下载与使用

1. 到 [**Releases**](../../releases) 下载最新的发布压缩包，解压到任意文件夹（路径建议纯英文、无空格 / 特殊字符）。
2. 申请 DashScope API Key：<https://bailian.console.aliyun.com/?apiKey=1#/api-key>（形如 `sk-xxxx`）。
3. 用记事本打开解压目录里的 `config.yaml`，把
   ```yaml
   dashscope_api_key: ""
   ```
   双引号里填上你的 Key，保存。**只需填一次。**
4. 把待整理的 `.xlsx`（里面是嵌入单元格的门店照片）拖到 `运行工具.bat` 上，或双击 `运行工具.bat` 按提示操作。
5. 跑完在 `output\` 目录取两个成品：
   - `陈列整理结果.xlsx` —— 主成品（按门店 × 档期归位的分类结果）
   - `陈列原图汇总.xlsx` —— 核对参考表（去重/丢弃/溢出的原图）

详细图文步骤见解压包内的 `说明书.md`。

## 费用与隐私说明

- 分类使用阿里云通义千问 `qwen-vl-max` 视觉模型，**按调用量计费，费用进你自己的阿里云账户**。
- 处理时图片会**上传到阿里云通义 API** 做识别。介意数据出境 / 隐私的，请评估后再使用。
- 本工具不收集、不上传任何数据给本项目作者；API Key 只保存在你本地的 `config.yaml`。
